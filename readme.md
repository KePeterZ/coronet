# Coronet
This is the repo for Coronet, the all-in-one Covid-19 application.

## Dependencies
* node.js
* express.js
* ejs
* nodemon (optional)

## Installing

```
git clone https://github.com/KePeterZ/coronet
npm install
```

## Using

```
npm start
```

## Default options
* Static files should be placed in ./static/[whatever]
* ejs files are in ./views/
