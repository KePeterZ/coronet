# Hello World
This is a Hello World example, written in Node.js using express, ejs, and MongoDB.

## Dependencies
* node.js
* express.js
* ejs
* nodemon (optional)

## Installing

```
git clone https://github.com/KePeterZ/helloworld.js
npm install
```

## Using

```
npm start
```

## Default options
* Static files should be placed in ./static/[whatever]
* ejs files are in ./views/
